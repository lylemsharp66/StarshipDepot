# StarshipDepot
A orbital mechanics teaching game utilizing Spacex Starship specs and realistic orbital mechanics to accomplish a handful of planetary missions. Vibe coded with Gemini 3.5 and Claude 4.6.

Starship_depot_v2 is an orbital mechanics educational game that simulates missions to the moon, mars and beyond utilizing Spacex Starship based on published specs and the ability to refuel in orbit. Titan requires a fuel generation capability in order for most return trips, but all other destinations already have fuel depots deployed.

Place the HTML file in a single directory and open starship_depot_v3.html in the javascript capable web browser of your choice. Ages 7-12 are probably best suited.
